## 3. Projenin Amacı ve Kısa Açıklaması

Bu projenin amacı, kullanıcının yüklediği anime karakter görselini analiz ederek karakter hakkında tahmin, profil ve karşılaştırma raporu oluşturmaktır.

Uygulama; anime karakter görselleri üzerinden çalışan bir görsel analiz ve karakter karşılaştırma projesidir. Kullanıcı bir anime görseli yüklediğinde sistem görseli işler, modelden gelen etiketleri değerlendirir, tarama akışını gösterir ve karakter raporu oluşturur.

Projede karakter tahmini, görsel özellik çıkarımı, karakter profili oluşturma, karakter bilgilerini yerel veritabanından gösterme ve birden fazla karakteri karşılaştırma özellikleri bulunmaktadır.

Tam çalışan proje paketi içinde model dosyası, etiket dosyası, ONNX Runtime Web dosyaları, yerel karakter veritabanı ve çalıştırma dosyaları birlikte yer almaktadır. Bu nedenle tam paket indirildiğinde uygulama yerel olarak çalıştırılabilir.

## 4. Kullanılan Teknolojiler / Kütüphaneler

Projede kullanılan teknolojiler ve kütüphaneler aşağıdaki gibidir:

- **HTML5:** Uygulamanın sayfa yapısı için kullanıldı.
- **CSS3:** Arayüz tasarımı, koyu yeşil tema, kart yapıları ve responsive görünüm için kullanıldı.
- **JavaScript:** Görsel yükleme, model hazırlama, analiz akışı, karakter raporu ve karşılaştırma sistemi için kullanıldı.
- **Vanilla JavaScript:** Ek bir frontend framework kullanılmadan saf JavaScript ile geliştirildi.
- **Python:** Yerel HTTP sunucusu ve başlatıcı sistemi için kullanıldı.
- **ONNX Runtime Web:** ONNX modelinin tarayıcı üzerinde çalıştırılması için kullanıldı.
- **WebAssembly / WASM:** ONNX Runtime Web çalışma dosyaları için kullanıldı.
- **ONNX Model:** Anime görsellerinden etiket tahmini almak için kullanıldı.
- **CSV:** Model etiketlerinin tutulduğu `selected_tags.csv` dosyası için kullanıldı.
- **JSON:** Karakter bilgilerini tutan `characters_db.json` dosyası için kullanıldı.
- **LocalStorage:** Analiz geçmişini tarayıcıda saklamak için kullanıldı.
- **BAT Dosyaları:** Windows üzerinde uygulamayı kolay başlatmak ve durdurmak için kullanıldı.

## 6. Kurulum Adımları

Projeyi kurmak için aşağıdaki adımlar izlenmelidir:

1. GitHub proje bağlantısı açılır.
2. GitHub Releases bölümünden tam çalışan proje paketi indirilir.
3. İndirilecek tam paket dosyası:

```text

```

4. İndirilen ZIP dosyası bilgisayarda bir klasöre çıkarılır.
5. Klasör çıkarıldıktan sonra aşağıdaki dosyaların bulunduğu kontrol edilir:


6. Paket içinde `portable_python/` klasörü bulunduğu için ayrıca Python kurulumu gerekmeden çalıştırılması hedeflenmiştir.
7. Proje doğrudan `index.html` dosyasına çift tıklanarak değil, yerel sunucu üzerinden çalıştırılmalıdır.

## 7. Çalıştırma / Kullanım Talimatları

Projeyi çalıştırmak için en kolay yöntem:

1. İndirilen proje paketi ZIP'ten çıkarılır.
2. Ana klasördeki `çalıştır.bat` dosyasına çift tıklanır.
3. Yerel sunucu otomatik olarak başlatılır.
4. Tarayıcı otomatik olarak açılır.
5. Uygulama yerel adreste çalışır.

Alternatif yöntem:

1. `PersonaSense_AI_Mini_Final` klasörüne girilir.
2. `Baslat_PersonaSense.bat` dosyasına çift tıklanır.
3. Uygulama tarayıcıda açılır.
4. Uygulama genellikle aşağıdaki adreste çalışır:

```text
http://127.0.0.1:8011/?launcher=1
```

Port 8011 doluysa uygulama 8012-8020 arası uygun bir yerel port üzerinden açılabilir.

### Kullanım Akışı

1. Uygulama açıldığında ana sayfa görüntülenir.
2. Kullanıcı analiz etmek istediği anime karakter görselini yükler.
3. Uygulama dosya türünü ve dosya boyutunu kontrol eder.
4. ONNX modeli hazırlanır.
5. Görsel model üzerinden analiz edilir.
6. Tarama akışı ekranında görselden çıkarılan temel özellikler gösterilir.
7. Karakter raporu ekranında tahmin edilen karakterler ve karakter bilgileri listelenir.
8. Karakter karşılaştırma ekranında karakterler ikili şekilde karşılaştırılabilir.
9. İstenirse rapora yeni karakter eklenebilir.
10. Uygulama kapatılmak istendiğinde `Durdur_PersonaSense.bat` dosyası kullanılabilir.
